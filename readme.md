How to use

1. Open the Menu

Click the menu button (☰ or “Menu”) on the screen

This opens the emulator control panel

2. Load Your ROM of choice

Click Load ROM

Select your folder or .zip file from your computer

Example files:

Pokemon Emerald.zip

Legend of Zelda Ocarina of Time.zip

The game will not start yet if the core isn’t chosen

3. Select the Emulator Core (IMPORTANT)

After loading the ROM, find Core / Emulator

Choose whichever core represents the console you want to play. 

4. Start the Game

Click Start or Play

Your game will begin normally

controls may vary from console to console 

You can change these under:

Menu → Controls

Saving Your Game (Very Important)
In-game saves (recommended)

Use the game’s normal save system

Save States (optional)

Menu → Save State

Menu → Load State

⚠️ Save states may not always persist if you clear browser data.

Best Settings for WebRetro (Optional)

If available in your menu:

Audio latency: Low

Video scaling: Integer / Pixel Perfect

VSync: off 

Use Chrome for best support
